# ts-flow

> Workflow orchestration in TypeScript

## Status

This package is a work in progress and not yet published. Expect breaking changes.

## Usage

Install bun: <https://bun.sh>

Install dependencies:

```bash
bun install
```

Run test:

```bash
bun test
```
